Verificación de Configuración Appwrite

Instrucciones:

  1. Abre las herramientas de desarrollador (F12)
  2. Ve a la pestaña Console
  3. Recarga esta página
  4. Busca mensajes que empiecen con "✅" o "⚠️"
  5. Si ves "⚠️ APPWRITE_ENDPOINT está vacío!", las variables no se pasaron durante la compilación

Para compilar correctamente:

flutter build web --release --dart-define="APPWRITE_ENDPOINT=https://api.lauraydaniel.es/v1" --dart-define="APPWRITE_PROJECT_ID=6938ac61002541a7230e" --dart-define="APPWRITE_API_KEY=..." --dart-define="APPWRITE_DATABASE_ID=boda" --dart-define="APPWRITE_RSVP_COLLECTION_ID=rsvps" --dart-define="APPWRITE_GALLERY_COLLECTION_ID=gallery_photos" --dart-define="APPWRITE_STORAGE_ID=6938b171003af1f91b94"

O ejecuta: .\compilar.bat

Verificar en Network:

  1. Ve a /formulario
  2. Abre F12 → Network
  3. Llena y envía el formulario
  4. Busca una petición que contenga api.lauraydaniel.es
  5. Si ves "rsvps" sin dominio, las variables están vacías
